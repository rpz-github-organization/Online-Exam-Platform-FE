## 接口文档范例

### 登录 

##### 接口功能

用于已有用户的登录功能

##### 请求方式

`GET`

##### URL

`/login`

##### 请求数据格式

```json
{
 	uid: "2018110211",
	password: "bbac9779"
}
```

##### 参数

|   参数   | 必选 |  类型  |     说明      |
| :------: | :--: | :----: | :-----------: |
|   uid    | true | string | 用户学号/工号 |
| password | true | string |   用户密码    |

##### 返回数据格式

```json
{
  errCode: 100404,
  msg: "该用户不存在",
 	data: {
    status: 
  }
}
```
